# Games Challenge
  
## Overview  
 This backend code is a challenge done in Rocketseat Ignite NodeJS Bootcamp.
 
## Instalation & Running
Since this was a challenge, the only purpose of this repository is to check my "repositories" methods

You can do:
```shell
yarn
```
or:
```shell
npm install
```

then to run the tests
```shell
yarn test
```
